# Get Started With Unity

This project is part of my journey to understand and explore the Unity Engine, its core tools, and workflow. It serves as a hands-on introduction to Unity development while also practicing version control using Fork.

## Project Overview

The goal of this project is to:
- Learn the basics of Unity (scenes, game objects, components)
- Understand how Unity’s editor and tools work
- Practice project structure and game logic fundamentals
- Use Fork as a Git client for managing version control

Every update to this project has been pushed using Fork, helping me gain real-world experience with Git workflows.

## Tools & Technologies

- Unity Engine  
- C#  
- Fork (Git client)  
- GitHub (repository hosting)  

## Version Control

All commits and updates are managed through Fork.  
This project demonstrates:
- Creating repositories  
- Cloning and managing projects  
- Committing and pushing changes  
- Maintaining version history
